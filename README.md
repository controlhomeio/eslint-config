# eslint-config
Personal ESLint config

<br/>

### Installation

```sh
npm i @controlhome/eslint-config eslint eslint-plugin-unused-imports @typescript-eslint/eslint-plugin --save-dev
```

<br/>

### Usage

Create an .eslintrc.json with the following content:

```js
{
  "extends": "@controlhome"
}
```
